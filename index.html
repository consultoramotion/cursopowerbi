Módulo 1: Introducción a Power BI

1.5 Horas

¿Qué es Power BI? ¿Cómo se vincula con el resto de la suite Power Platform? Casos de uso de la industria.

Temario Detallado

  • Contexto de Power Platform: Power BI, Power Apps, Power Automate
  • Qué es Power BI
  • Componentes de Power BI: Desktop, Service, Mobile
  • Arquitectura y licenciamiento: Free, Pro, Premium
  • Casos de uso por industria: retail, finanzas, manufactura, salud
  • Instalación y exploración de la interfaz

Módulo 2: Conexión a Orígenes de datos

1.5 Horas

Evaluar diferentes tipos de orígenes de datos, desde archivos Excel hasta bases de datos empresariales y servicios en la nube.

Temario Detallado

  • Tipos de orígenes: archivos, bases de datos, servicios en línea, APIs
  • Modos de conectividad: Import, DirectQuery, Live Connection
  • Conexión a Excel y archivos locales
  • Conexión a SQL Server y bases de datos relacionales
  • Conexión a listas de SharePoint
  • Conexión a carpetas de archivos

Módulo 3: Transformación en Power Query

4 Horas

Power Query es el motor ETL que permite limpiar, transformar y preparar datos de forma reproducible usando el lenguaje M. Dominar estas técnicas es fundamental.

Temario Detallado

  • Interfaz de Power Query Editor: paneles, pasos aplicados, tipos de datos
  • Transformaciones básicas: filtrado, eliminación, reordenamiento de columnas
  • Transformaciones de formato: texto, números, fechas
  • Operaciones avanzadas: split, merge, pivot/unpivot
  • Columnas condicionales y personalizadas con lenguaje M
  • Combinar consultas: merge (joins) y append (uniones)
  • Tipos de joins: inner, left, right, full outer, anti
  • Agrupación y agregación de datos
  • Parámetros y funciones personalizadas
  • Query folding y optimización de rendimiento
  • Mejores prácticas: organización, documentación, eficiencia

Módulo 4: Diseño del modelo de datos

1.5 Horas

El modelo de datos es el núcleo de Power BI que determina el rendimiento y funcionalidad. Un diseño óptimo sigue principios de modelado dimensional.

Temario Detallado

  • Conceptos: tablas de hechos vs dimensiones, esquema estrella
  • Configuración de relaciones: cardinalidad, direccionalidad de filtros
  • Relaciones activas e inactivas
  • Tabla calendario: creación y configuración como tabla de fechas
  • Mejores prácticas de modelado

Módulo 5: Visualizaciones y Tableros

4 Horas

Creación de gráficos, tarjetas, segmentadores y reportes interactivos para facilitar el análisis y la toma de decisiones estratégicas.

Temario Detallado

  • Tipos de visualizaciones: tablas, matrices, barras, líneas, mapas, KPIs
  • Configuración de campos: ejes, valores, leyendas, tooltips
  • Formato y personalización: colores, títulos, etiquetas, temas
  • Segmentaciones y filtros: configuración y sincronización
  • Interactividad: drill-down, drill-through, filtrado cruzado
  • Bookmarks y botones para navegación
  • Diseño de página: alineación, distribución, mobile layout
  • Buenas prácticas para dashboards ejecutivos

Módulo 6: Cálculos con DAX

2 Horas

DAX (Data Analysis Expressions) es el lenguaje de fórmulas que permite crear medidas y cálculos avanzados como comparaciones temporales y rankings.

Temario Detallado

  • Medidas, columnas, tablas calculadas
  • Contexto de fila y contexto de filtro
  • Funciones de agregación, filtro, relación, reglas lógicas
  • CALCULATE: modificación del contexto de filtro

Módulo 7: Power BI Service

1 Hora

Power BI Service es la plataforma en la nube para publicar, compartir y consumir informes. Se aprende el proceso completo de despliegue.

Temario Detallado

  • Publicación desde Desktop a Service
  • Espacios de trabajo: roles y permisos
  • Actualización de datos: manual y programada
  • Creación de paneles: anclar visualizaciones
  • Q&A: preguntas en lenguaje natural
  • Monitoreo de uso y métricas

Módulo 8: Conclusiones

0.5 Horas

Consolidación de conceptos clave y orientación sobre próximos pasos para continuar desarrollando habilidades en Power BI.

Temario Detallado

  • Resumen del flujo completo: conectar, transformar, modelar, visualizar, publicar
  • Resumen de buenas prácticas
